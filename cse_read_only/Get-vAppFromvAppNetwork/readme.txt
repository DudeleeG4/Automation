Get-vAppfromvAppNetwork.ps1

This script is intended to make it easier to find which vApp a vApp network is a part of.
It will return the vApp Network name and the vApp name. It will also provide an option to retrieve the Org.
The script will loop through, prompting the user for another vApp network unless they specify that they do not want to continue.

To run the script, simply right-click and select "Run with powershell"
