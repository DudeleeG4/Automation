Accenture Storage Policies

This script will retrieve all of Accenture's VMs from the vCenter and list their default storage policy + the actual storage policy of each disk on the VMs.

To run the script - go on the IL2 farnborough jumpbox ajw00003i2 then simply right-click the script and select "Run with powershell". 

The script will output a .csv file at "C:\Scripts\Technology\CSE\Accenture Storage Policies.csv"
